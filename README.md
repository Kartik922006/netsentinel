# 🔐 NetSentinel – AI Network Port Scanner & Intrusion Detection System

NetSentinel is a Flask-based cybersecurity web application that scans network ports of a target IP address and detects potential security risks. It helps in basic intrusion detection and network analysis using socket programming.

---

## 🚀 Live Demo

🌐 https://netsentinel-mjgk.onrender.com

---

## 📌 Features

* 🔍 Scan common network ports
* ⚡ Detect open and closed ports
* 🚨 Highlight high-risk ports (FTP, Telnet, SMB)
* 📊 Risk classification (Low / Medium / High)
* 💻 Clean and responsive UI
* 📄 Download scan report

---

## 🛠 Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS
* **Networking:** Python Socket Programming
* **Deployment:** Render

---

## ⚙️ How It Works

1. User enters a target IP address
2. Application scans predefined ports using socket connection
3. Identifies open/closed ports
4. Flags risky ports
5. Displays overall risk level
